# AutoCommit GitGenius

Welcome to AutoCommit GitGenius VSCode Plugin! This extension helps streamline your development workflow by automating Git commits.

## Features

* Automates your commit messages smartly and efficiently.
* Integrates directly into your Visual Studio Code environment.

## Requirements

* Visual Studio Code version 1.80.0 or higher.
* An active Git repository.
